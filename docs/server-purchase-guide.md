# 云服务器购买指南

## 阿里云购买步骤（推荐）

### 1. 注册账号
访问：https://www.aliyun.com
- 使用手机号注册
- 实名认证（需要身份证）

### 2. 购买ECS云服务器

#### 方式一：新用户优惠（最便宜）
1. 访问：https://www.aliyun.com/daily-act/ecs/activity_selection
2. 选择"新手入门"或"个人试用"
3. 通常有3个月免费或¥99/年的优惠

#### 方式二：正常购买
1. 访问：https://www.aliyun.com/product/ecs
2. 点击"立即购买"
3. 配置选择：

**基础配置**：
```
地域：选择离您最近的区域
  - 华东（杭州）/ 华北（北京）/ 华南（深圳）

实例规格：
  - 2核4GB（ecs.t6-c1m2.large 或类似）

操作系统：
  - Ubuntu 22.04 64位
  - 不要选择Windows（贵且不需要）
```

**网络和安全组**：
```
网络类型：专有网络（VPC）
带宽计费：按固定带宽
带宽值：3 Mbps
```

**系统配置**：
```
登录凭证：自定义密码
  - 设置root密码（记住这个密码！）
```

**分组设置**：
```
实例名称：member-management
```

4. 确认订单，支付

### 3. 购买完成后

#### 获取服务器信息
1. 登录阿里云控制台
2. 进入"云服务器ECS"
3. 查看实例列表
4. 记录以下信息：
   - **公网IP地址**（重要！）
   - 实例ID
   - 地域

#### 配置安全组
1. 在ECS控制台，点击"安全组"
2. 配置入方向规则：
```
端口    授权对象      描述
22      0.0.0.0/0    SSH登录
80      0.0.0.0/0    HTTP
443     0.0.0.0/0    HTTPS
8000    0.0.0.0/0    后端API（可选，用于调试）
```

## 腾讯云购买步骤

### 1. 注册账号
访问：https://cloud.tencent.com
- 微信扫码注册（很方便）
- 实名认证

### 2. 购买CVM云服务器

#### 方式一：新用户优惠
1. 访问：https://cloud.tencent.com/act/cps/redirect
2. 查找"新人专享"活动
3. 通常有轻量服务器特惠

#### 方式二：正常购买
1. 访问：https://cloud.tencent.com/product/cvm
2. 点击"立即购买"
3. 配置选择：

**基础配置**：
```
地域：选择离您最近的
机型：2核4GB
镜像：Ubuntu 22.04 LTS
```

**存储和带宽**：
```
系统盘：40GB SSD
带宽：3Mbps
```

4. 设置登录密码（重要！）
5. 确认购买

### 3. 购买完成后
- 记录公网IP地址
- 配置防火墙规则（同阿里云安全组）

## 购买后验证

### SSH连接测试
在本地电脑打开终端/PowerShell：

```bash
# 替换为您的服务器IP
ssh root@your_server_ip

# 输入购买时设置的密码

# 连接成功后，会看到类似内容：
# Welcome to Ubuntu 22.04 LTS...
root@iZxxxxxx:~#
```

### 更新系统
```bash
# 连接服务器后执行
sudo apt update && sudo apt upgrade -y
```

## 注意事项

1. **密码安全**：
   - 使用强密码
   - 不要泄露给他人

2. **地域选择**：
   - 选择离目标用户近的地区
   - 国内用户选华东/华北/华南

3. **续费提醒**：
   - 设置自动续费或到期提醒
   - 新用户优惠结束后价格会上涨

4. **备份**：
   - 定期创建快照
   - 重要数据要备份
