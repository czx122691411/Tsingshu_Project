# 域名购买指南

## 域名选择建议

### 好域名的标准
1. **短小**：越短越好，容易记忆
2. **易拼**：避免生僻字和多音字
3. **有意义**：与业务相关
4. **后缀合适**：.com > .cn > .net

### 推荐域名示例

**书店会员系统**：
- 英文：bookstore.com, library.com, reading.com
- 拼音：shudian.com, tushuguan.cn, duzhe.cn
- 组合：yourname-book.com, cityreading.cn

**注意**：好的域名可能已被注册，需要多尝试几个。

### 阿里云万网购买（推荐）

#### 1. 查询域名
访问：https://wanwang.aliyun.com

在搜索框输入想要的域名，如：
```

```

1. 进入域名清单
2. 选择购买年限（建议1年起）
3. 填写域名持有者信息（必须真实）
4. 支付完成

## 购买后配置

### 1. 域名实名认证（必须）
国内域名必须实名认证：
1. 进入域名控制台
2. 上传身份证照片
3. 等待审核（1-3个工作日）

### 2. 域名解析（DNS配置）

#### 添加解析记录

在域名控制台，添加以下解析记录：

**记录1：主域名**
```
记录类型：A
主机记录：@
记录值：你的服务器公网IP
TTL：10分钟
```

**记录2：www子域名**
```
记录类型：A
主机记录：www
记录值：你的服务器公网IP
TTL：10分钟
```

**记录3：api子域名（可选，推荐）**
```
记录类型：A
主机记录：api
记录值：你的服务器公网IP
TTL：10分钟
```

### 3. 解析生效验证

在本地电脑执行：
```bash
# Windows
ping yourdomain.com
ping www.yourdomain.com

# Mac/Linux
ping yourdomain.com
```

如果返回你的服务器IP，说明解析成功。

解析生效时间：通常10分钟-1小时

## 域名管理建议

### 安全设置
1. **开启域名锁**：防止域名被恶意转移
2. **开启WHOIS保护**：隐藏个人信息（可选）
3. **设置自动续费**：防止域名过期

### 续费提醒
- 设置日历提醒到期前1个月
- 或开启自动续费

## 常见问题

### Q：域名被注册了怎么办？
A：
1. 换个后缀试试（.com → .cn）
2. 添加前缀/后缀
3. 使用缩写

### Q：需要买多个后缀吗？
A：不必，一个.com就够了

### Q：解析不生效怎么办？
A：
1. 检查记录值是否正确
2. 等待更长时间（最长24小时）
3. 清除本地DNS缓存
